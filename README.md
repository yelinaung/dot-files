Dot-files
===========
My (new) dot-files configuration for vim and tmux.


Installation
------------
```bash
$ git clone https://github.com/yelinaung/dot-files.git
$ cd dot-files
$ sh install.sh
```


Install Plugins:
---------------

Launch `vim` and run `:PluginInstall`

To install from command line: `vim +PluginInstall +qall`
